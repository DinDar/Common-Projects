 Error #10 - Ошибка IOException при попытке создать файл, или записать его. 
 			По каким-то причинам невозможно создать файл, в который будет произведена сереализация. 
 			Возможно у программы нету прав для создания файлов в файловой системе, попробуйте запустить 
 			программу в режиме администратора (для Windows), или под root пользователем (для Unix 
 			подобных систем);
 
 Error #11 - Ошибка FileNotFoundException при попытке работы с файлом.
 			Библиотека позволяет создавать файлы с кастомными именами. По какой-то присине ей не 
 			удалось создать файл с указанным вами именем. Возможно вы использовали некоректные символы
 			для названия (Используйте латниские буквы A-Z и цифры). Или-же по каким-то причинам библи-
 			отека не может найти необходимый файл, возможно вы указали имя этого файла неправильно
 			(Если ошибка произошла при попытке сереализации).
 			
 Error #12 - Ошибка ClassNotFoundException при попытке десереализовать файл.
 			Сереализуемый файл не является классом, который библиотека может распознать. Возможно в
 			класс, который вы хотите десереализовать были внесены изменения (имеется в виду в код),
 			попробуйте сереализовать класс опять.
